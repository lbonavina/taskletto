✦ Novidades

O que há de novo na v2.0

Subtarefas, sync com GitHub, atalhos no sidebar e UX repensada — a maior atualização do Taskletto até hoje.

✦ Novo
✅
Subtarefas
Quebre tarefas complexas em etapas menores. Cada tarefa agora pode ter subtarefas independentes, com progresso calculado automaticamente e exibido na lista principal.
Subtarefas aninhadas Progresso automático Visível na lista
✦ Novo
🔗
GitHub Sync
Sincronize suas tarefas e notas com um Gist privado no GitHub. Seus dados ficam versionados e acessíveis de qualquer dispositivo, sem depender de nenhum servidor próprio.
Backup automático Versionado Gist
✦ Novo
⭐
Atalhos no sidebar
Fixe tarefas e notas no menu lateral para acessá-las com um clique. Adicione pelo botão ⭐ na página ou direto na lista — o label atualiza automaticamente quando você renomeia o item.
Tasks e notas localStorage Label sincronizado
⚡ Melhorado
⚡
Início instantâneo
Cache de configurações, rotas e views no startup. O app abre significativamente mais rápido — sem tela de loading, direto para as suas tarefas.
Config cache Route cache View cache
⚡ Melhorado
🎨
UI/UX melhorada
Dashboard com saudação dinâmica por período do dia, editor de notas com novos recursos de formatação, instalador Windows atualizado e dezenas de ajustes visuais em todo o app.
Dashboard redesenhado Editor melhorado Novo instalador
v1.3 — versão anterior
✦ Novo
📣
Callout Blocks
Caixas de destaque estilo Notion diretamente no editor de notas. Disponíveis via menu slash (/) ou pelo novo botão na toolbar, com 6 tipos: Info, Sucesso, Aviso, Perigo, Dica e Nota.
ℹ️ Info ✅ Sucesso ⚠️ Aviso 🚨 Perigo 💡 Dica 📝 Nota
v1.2 — versão anterior
✦ Novo
🔁
Recorrência de tarefas
Tarefas que se repetem automaticamente. Ao concluir uma tarefa recorrente, a próxima ocorrência é criada na hora.
Diária Semanal Mensal
✦ Novo
⏱️
Time tracking
Timer start/stop direto na sidebar da tarefa. Defina o tempo estimado e acompanhe o progresso com uma barra visual.
Start / Stop Tempo estimado Barra de progresso
✦ Novo
📊
Dashboard redesenhado
KPIs com destaque visual, seção "Hoje" com as tarefas do dia, streak de produtividade e gráfico com tooltip interativo.
KPIs Hoje Streak
✦ Novo
💾
Export / Import JSON
Faça backup de todas as suas tarefas e notas em um arquivo JSON ou migre seus dados entre dispositivos com um clique.
Backup Migração JSON
✦ Funcionalidades

Tudo que você precisa,
nada que não precisa.

Sem bloat, sem assinatura. Só o essencial para manter o foco no que importa.

📋
Gestão de tarefas completa
Status, prioridade, vencimento, categorias e descrição. Tudo em um lugar com filtros rápidos e busca.
✅
Subtarefas
Quebre qualquer tarefa em etapas menores. O progresso é calculado automaticamente e exibido na lista.
🔗
GitHub Sync
Sincronize tarefas e notas com um Gist privado. Backup automático, versionado, sem servidor próprio.
⭐
Atalhos no sidebar
Fixe qualquer tarefa ou nota no menu lateral para acesso imediato. O label atualiza sozinho ao renomear.
📝
Editor de notas rico
Baseado no Tiptap — negrito, itálico, tabelas, checklists, blocos de código, callouts e muito mais.
📣
Callout blocks
Caixas de destaque estilo Notion. 6 tipos com cores e ícones, inseríveis via slash (/) ou toolbar.
🔁
Recorrência automática
Configure tarefas para repetirem diária, semanal ou mensalmente. A próxima ocorrência é criada automaticamente.
⏱️
Time tracking integrado
Timer embutido por tarefa com tempo estimado e barra de progresso visual.
⚡
Início instantâneo
Cache no startup. O app abre mais rápido, sem tela de loading — direto para o seu fluxo.
🏷️
Categorias com emoji e cor
Organize tarefas em categorias visuais com ícone emoji personalizado e cor de identificação.
🌙
Tema claro e escuro
Alterne entre dark e light mode com um clique. O tema é salvo entre sessões.
🔒
100% local e offline
Seus dados ficam no seu computador. Sem conta, sem servidor, sem nada sendo enviado.
✦ Como funciona

Pronto em 3 passos.

Sem cadastro, sem configuração complicada. Baixa e usa.

01
Baixe o instalador
Clique em Download, escolha Windows e execute o instalador. Leva menos de um minuto.
→
02
Crie suas categorias
Defina categorias com cores e emojis para organizar tarefas por projeto, área ou contexto.
→
03
Comece a produzir
Adicione tarefas, escreva notas e use os filtros para focar no que precisa ser feito agora.

Baixe agora. É grátis.
